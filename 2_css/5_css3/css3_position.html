css3 속성

position 속성

static

고정 위치(기본 속성)

  • 기본 속성이므로 별도로 설정하지 않음
  • 다른 속성에서 기본 속성으로 되돌릴 때 사용

relative

상대적인 위치

  • Z값을 띄워놓은 상태 (Z값이 높아짐)
  • top: left: right: bottom: 등 위치값 사용 가능
  • 위치값의 기준은 처음 위치

하위 absolute

직계 상위요소 중 position 속성을 가진 요소가 있으면 그 요소를 기준으로 삼음

absolute

절대적인 위치

  • Z값을 띄워놓은 상태 (Z값이 높아짐)
  • top: left: right: bottom: 등 위치값 사용 가능
  • 위치값의 기준은 body태그 기준
    -단, right: bottom: 의 기준은 viewport
  • 영역을 차지하지 않음(float과 비슷)

fixed

절대적인 위치

  • Z값을 띄워놓은 상태 (Z값이 높아짐)
  • top: left: right: bottom: 등 위치값 사용 가능
  • 위치값의 기준은 Viewport 기준
  • 영역을 차지하지 않음(float과 비슷)
  • 스크롤값 고정 - Viewport의 위치가 고정이므로
