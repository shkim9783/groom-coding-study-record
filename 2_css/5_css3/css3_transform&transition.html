transform & transition

transform은 변형 / transition은 과정

transform 속성

translate

현재 위치 기준 / 이동
  • transform: translate(X값, Y값);
  • transform: translateX() translateY(); 둘로 나우어 사용 가능
  • 마이너스(-) 값 사용 가능

    • rotate

    중심 기준 / 시계방향 회전
  • transform: rotate(수치deg);
  • deg(degree) 단위 사용
  • 마이너스(-) 값 사용 가능

    • scale

    중심 기준 / 확대, 축소
  • transform: scale(배수); 혹은
    transform: scale(X배수,Y배수);
  • transform: scaleX() scaleY(); 둘로 나누어 쓸 수 있음
  • 기준은 1(==100%)
  • 마이너스(-) 값 사용 가능

    • skew

    중심 기준 / 기울어짐
  • transform: skew(X값,Y값);
    - 값을 하나만 입력하면 X값으로 인식
  • transform: skewX() skewY(); 둘로 나누어 사용 가능
  • deg(degree) 단위 사용
  • 마이너스(-) 값 사용 가능

    • position과 transform을 이용한 블록요소의 중앙 정렬
