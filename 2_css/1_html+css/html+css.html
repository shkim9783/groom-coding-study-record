선택자 {속성: 속성값;}
- 객체{무엇을: 어떻게;}


css를 html에 적용시키는 방법

1. Inline Style Sheet

  • html 문서 태그 안에 style 속성을 직접 입력
  • css 입력 내용이 매우 적을 때 사용
  • 우선 순위가 놆아 테스트 목적으로 주로 사용

2. Internal Style Sheet

  • html 문서의 head 영역 안에 style 태그를 작성하여 입력
  • 간단한 문서에 주로 사용

3. Linking Style Sheet

  • 별도의 css 파일을 작성하여 html 문서와 연결
  • head 영역 안에 link 태그를 입력한 후 link 태그 안에 href, rel 속성을 지정
  • 가장 일반적인 방법

선택자 유형

  1. tag(type) 선택자 - '태그명'을 그대로 입력
  2. class 선택자 - '.' + 클래스명 입력(클래스명 앞에 점)
  3. id 선택자 - '#'+'아이디명'을 입력 (아이디명 앞에 샵)
  4. 복합 선택자

선택자는 구체적일수록 우선순위가 높다
